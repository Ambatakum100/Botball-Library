Weleetka Botball Library Documentation

By Grant Hawkins

Installation

botballLibraryV2.h 

Prerequisites

This library is designed for use with the KIPR Wombat and includes various customizable functions to suit your specific needs.

1. Enabling Advanced Mode

To use this library, you must enable Advanced Mode on your Wombat:

  1. Connect your laptop or Chromebook to the Wombat homepage.
  2. Navigate to "User Preferences" on the homepage.
  3. Select your user profile and set it to Advanced Mode.

Enabling Advanced Mode allows you to add files beyond the default main.c, including the botballLibrary.h header file.

2. Adding the Library to Your Project

Once Advanced Mode is enabled, follow these steps to include the library in your project:

  1. Open your project in the Wombat interface.
  2. Under the "Include Files" section, select "Add New File".
  3. Click "Choose File" and upload botballLibrary.h.

After completing these steps, you will have full access to all the library's functions!

3. Hardware Requirements & Configuration

  • This library is designed for two-wheeled robots and is not optimized for mecanum wheels or other drive types.
  • Motor Ports:
  • Right Wheel: Port 3
  • Left Wheel: Port 0
  • Tophat Sensor (Black Line Sensor) Configuration:
  • The threshold for detecting black is 3800.
  • The squareUp() function requires two tophat sensors placed horizontally (parallel to the wheels).
  • Left Sensor: Connected to analog(1).
  • Right Sensor: Connected to analog(2).

Note: These values can be adjusted in the definitions section at the top of the library file if needed.

4. Speed Considerations

  • All movement functions use mav().
  • For optimal accuracy, keep speed values between -1000 and 1000.
  • Maximum speed values can range from -1500 to 1500, but accuracy may decrease at higher speeds.

Functions

1. stop();

Description:
Stop both wheels for half a second before turning them off.

2. drive(int distance, int speed);

Description:
Drive a specified distance in inches. Use a negative speed to move backward.

Example Usage:

drive(5, 1000);

3. driveET(int port, int distance, int speed);

Description:
Drives forward until the distance sensor reads the specified value.

Parameters:

  • port – The sensor port number.
  • distance – The target distance to stop at.
  • speed – The speed at which to drive.

Example Usage:

driveET(0, 2600, 1000);

4. turn(int degrees, int speed, char* direction);

Description:
Turns by a specified number of degrees at a given speed in the chosen direction.

Parameters:

  • degrees – The amount to turn.
  • speed – The speed of the turn.
  • direction – "left" or "right".

Example Usage:

turn(90, 1000, "right");

5. lineFollow(int distance, int speed, int port);

Description:
Follows the right side of a black line using a tophat sensor.

Parameters:

  • distance – Distance to follow the line (in inches).
  • speed – The speed of movement.
  • port – The sensor port number.

Example Usage:

lineFollow(12, 1000, 1);

6. cameraFollow(int port, int channel, int ET, int speed, int turnSpeed);

Description:
Drives toward an object using HSV blob tracking until the distance sensor detects the object is too close.

Parameters:

  • port – Distance sensor port.
  • channel – The color channel of the object (configured on the Wallaby).
  • ET – Distance at which the robot should stop.
  • speed – Driving speed.
  • turnSpeed – Speed at which the robot turns to re-center the object.

Example Usage:

cameraFollow(0, 0, 2600, 1000, 500);

7. cameraFollowDistance(int channel, int distance, int speed, int turnSpeed);

Description:
Drives toward an object for a specified distance using HSV blob tracking.

Parameters:

  • channel – The color channel of the object.
  • distance – The distance to drive (in inches).
  • speed – Driving speed.
  • turnSpeed – Speed at which the robot turns to re-center the object.

Example Usage:

cameraFollowDistance(0, 12, 1000, 500);

8. slowServo(int port, int position, int speed, int disable);

Description:
Moves a servo to a specified position at a controlled speed.

Parameters:

  • port – The servo port.
  • position – The target position of the servo.
  • speed – The speed at which the servo moves.
  • disable – Whether to turn off the servo at the end (0 = off, 1 = stay powered).

Example Usage:

slowServo(0, 500, 10, 0);

9. checkBattery();

Description:
Check the battery voltage and warn if it is too low. Currently broken, but still useful—pay attention if the voltage reads below 0.20.

Example Usage:

checkBattery();

10. driveStraight(int speed, double distance);

Description:
It uses a gyroscope and PID control for improved straight-line driving over long distances. Not recommended for short distances due to slight left-right jitter.

Parameters:

  • speed – Driving speed.
  • distance – Distance to drive (in inches).

Example Usage:

driveStraight(1000, 12);

11. squareUp(int threshold, int speed);

Description:
Reverses at a specified speed until two tophat sensors detect a black surface.

Parameters:

  • threshold – The value at which the sensor detects black.
  • speed – The speed at which the robot moves.

Example Usage:

squareUp(3800, 1000);

Final Notes:

  • Ensure sensor ports are properly configured before using any function.
  • Be mindful of sensor readings and voltage warnings, especially in autonomous movement functions.
